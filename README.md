# Philosophie-Delor

Dieses Repository enthält die Quellen für die persönliche Website **Philosophie-Delor**.

Die Website dient als zentraler Ort für:

* philosophische Essays und allgemeinverständliche Texte
* thematische Einführungen (u. a. Ethik, Epistemologie, Ontologie, Philosophiegeschichte)
* Verlinkungen zu wissenschaftlichen Veröffentlichungen (Preprints, Artikel, Projekte)

Der Schwerpunkt liegt auf philosophischer Arbeit mit wissenschaftlichem Anspruch sowie auf der Vermittlung philosophischer Inhalte für eine breitere Öffentlichkeit. Die Seite versteht sich nicht als Plattform für Meinungen oder Weltanschauungen, sondern als öffentlich zugängliche Schnittstelle zwischen Forschung, Reflexion und Vermittlung.

Weitere Projekte und Formate können zu einem späteren Zeitpunkt ergänzt werden.
